# Projekt: Smart-Repair-Matching (MVP)

Dieses Repository enthält den ersten funktionalen Prototypen für eine Plattform, die die Reparatur von Elektrogeräten vereinfacht und fördert.

## Kurzbeschreibung (Übung 12)

* [cite_start]**Problem:** Viele Konsumenten entsorgen defekte Elektronikgeräte vorzeitig, weil technisches Know-how fehlt, Reparaturanleitungen zu komplex sind oder der Zugang zu passenden Ersatzteilen und Experten fehlt[cite: 19, 41, 80].
* **MVP (Kernfunktionen):**
    * [cite_start]**Matching-Suche:** Nutzer können nach ihrem Gerät suchen, um gezielt Hilfe zu finden[cite: 49, 56].
    * [cite_start]**Teile- & Experten-Verzeichnis:** Anzeige von verfügbaren Ersatzteilen und lokalen Reparatur-Kontakten in Echtzeit[cite: 50].
    * [cite_start]**Interaktive Kontaktfunktion:** Über einen Button können Nutzer direkt Interesse an einer Reparatur bekunden oder Hilfe anfordern[cite: 81, 195].
* [cite_start]**Circular Flow:** Die App unterstützt primär den **"Slow"-Flow**, indem sie die Nutzungsdauer von Produkten durch Reparatur und Wiederverwendung verlängert ("Use longer" & "Use again")[cite: 11, 17, 72, 185].
* [cite_start]**Aktueller Stand:** Ein funktionsfähiger Web-Prototyp (MVP Phase 1) ist erstellt, dokumentiert und via GitHub Pages öffentlich zugänglich gemacht worden[cite: 55, 178, 209].

## Technische Umsetzung
* [cite_start]**Technologien:** HTML, CSS und JavaScript[cite: 192].
* [cite_start]**Deployment:** Veröffentlicht über GitHub Pages[cite: 182, 199].
* [cite_start]**Interaktivität:** Dynamische Filterung der Reparatur-Optionen und simulierter Anfrage-Prozess[cite: 211].
